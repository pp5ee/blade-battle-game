# 转刀刀 (Blade Blade) - 单机网页游戏

一款基于HTML5 Canvas开发的单机网页游戏，玩家控制角色收集刀刀，与NPC进行战略战斗，体验自由键盘移动的乐趣。

## 游戏特色

- **自由移动系统**: 使用WASD或方向键进行流畅的键盘移动
- **三色刀刀系统**: 红刀(最强) > 黄刀 > 蓝刀的颜色等级体系
- **战略战斗**: 基于刀刀数量和颜色的战斗计算
- **智能NPC**: 具有追逐、逃跑、回避等行为模式的敌人
- **游戏状态持久化**: 自动保存游戏进度

## 游戏玩法

### 基本操作
- **移动**: WASD 或 方向键
- **攻击**: 靠近敌人自动触发战斗
- **收集**: 接触刀刀自动收集

### 战斗系统
- **颜色等级**: 红刀(3倍) > 黄刀(2倍) > 蓝刀(1倍)
- **数量优势**: 刀刀数量越多，战斗力越强
- **战略回避**: 根据实力对比智能选择战斗或回避

### 刀刀系统
- **红刀**: 最强攻击力，稀有度高
- **黄刀**: 中等攻击力，平衡性佳
- **蓝刀**: 基础攻击力，获取容易

## 技术实现

### 核心架构
```
BladeGame (主游戏类)
├── Player (玩家角色)
├── Blade (刀刀实体)
├── NPC (非玩家角色)
└── GameState (游戏状态管理)
```

### 关键技术
- **HTML5 Canvas**: 2D游戏渲染
- **JavaScript ES6+**: 面向对象编程
- **requestAnimationFrame**: 60fps游戏循环
- **localStorage**: 游戏数据持久化
- **碰撞检测**: 圆形碰撞算法

### 物理系统
- **加速度/摩擦力**: 真实移动物理
- **速度限制**: 防止超高速移动
- **边界反弹**: 世界边界物理效果

## 游戏机制详解

### 移动系统
- **加速度控制**: 按键时间越长，速度越快
- **惯性滑行**: 松开按键后逐渐减速
- **方向指示**: 角色显示移动方向

### 刀刀收集
- **自动收集**: 接触即收集
- **视觉反馈**: 刀刀旋转和脉动效果
- **UI更新**: 实时显示刀刀数量

### NPC智能
- **行为模式**: 漫游、追逐、逃跑、回避、接近、环绕
- **实力评估**: 根据刀刀数量智能决策
- **战略移动**: 根据实力对比选择最佳策略

**NPC行为说明:**
- **漫游**: 随机移动探索世界
- **追逐**: 当玩家较弱时主动追击
- **逃跑**: 当玩家较强时快速逃离
- **回避**: 战略性躲避玩家攻击范围
- **接近**: 当NPC较弱时谨慎靠近
- **环绕**: 实力相当时围绕玩家移动

### 战斗计算
```javascript
// 战斗力量计算公式
power = (红刀数量 × 3 + 黄刀数量 × 2 + 蓝刀数量 × 1) × 衰减系数
```

### 游戏状态
- **自动保存**: 每30秒自动保存
- **页面关闭保护**: 关闭前自动保存
- **加载验证**: 检查存档时效性(1小时内有效)

## 故障排除

### 常见问题

**Q: 游戏无法启动**
A: 检查浏览器是否支持Canvas，确保JavaScript已启用

**Q: 移动不流畅**
A: 关闭其他标签页，确保浏览器有足够资源

**Q: 存档丢失**
A: 检查浏览器localStorage设置，避免隐私模式

**Q: NPC行为异常**
A: 刷新页面重新加载游戏状态

### 调试模式
在浏览器控制台中输入以下命令查看游戏状态：
```javascript

console.log('玩家位置:', game.player.x, game.player.y);

console.log('NPC数量:', game.npcs.length);
```
